Card Games

Uno is a classic, fun card game. It only requires 2 players, at minimum, and can go well into the night. Many versions of the game exists; one version allows you to even make your own Wild cards!

Monopoly Deal: The Card Game is a 15-minute card game where 2 or more players have to get as much property, as possible. Unlike the actual board game, which can be "never-ending", this quick-paced game will give you the same great feeling of being a property tycoon, in 15 minutes or less.
